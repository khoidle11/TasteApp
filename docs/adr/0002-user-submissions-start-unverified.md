# User submissions start unverified

TasteApp will allow users to create Restaurants, Locations, Canonical Dishes, Menu Items, RestaurantDishes, and Dish Reviews freely, but those records start unverified. AI Verification Assist can suggest duplicates, canonical names, misspellings, suspicious submissions, and merge candidates, but the product should treat AI output as evidence rather than automatic truth so messy user data does not permanently corrupt dish rankings.
